# fast & simple Hands-on environment for learning/praticing web security testing.
- Follow teachings at : [https://portswigger.net/web-security/all-labs](https://portswigger.net/web-security/all-labs)
- Add routes in project to practice learnings 

Clone the project
```bash
git clone https://github.com/FredericGariepy/Web_Security.git
```

Get in the project directory
```bash
cd Web_Security
```

(Optional) Virtualenv
```cmd
python -m venv venv
```

(Optional) Windows, activate Python virtual environment.
```cmd
venv\Scripts\activate
```

(Optional) macOS/Linux, activate Python virtual environment.
```bash
source venv/bin/activate
```

Install requirements (Flask & Flask-related)
```bash
pip install -r requirements.txt
```

Run
```bash
python main.py
```

Open a web browser and Go to: 
```
http://localhost:5000
```
